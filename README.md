# Система мониторинга температуры

## Описание
Программа для считывания температуры с устройства через последовательный порт и логирования данных. Включает:
1. Основную программу для чтения и логирования
2. Программу-симулятор устройства для тестирования

## Требования
- C++17 компилятор
- CMake 3.10+
- Для Windows: MinGW
- Для Linux: g++ и библиотеки разработки
